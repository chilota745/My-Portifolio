# 🌐 Portfolio Website (HTML, CSS & JavaScript)

A modern, minimalistic, and colorful personal portfolio website built using **HTML, CSS, and JavaScript**. This project is designed to showcase your skills, projects, and contact information in a clean and professional way.

---

## Features

* Minimalistic, sleek, and colorful UI design
* Fully responsive (mobile, tablet, desktop)
* About Me section
* Projects showcase section
* Skills section
* Contact form / social links
* Smooth scrolling and simple animations

---

## Tech Stack

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout (Flexbox/Grid)
* **JavaScript (Vanilla JS)** – Interactivity and dynamic behavior

---

## Design Guide

This project follows a **minimalistic, sleek, and colorful aesthetic**:

* Use **clean layouts** with lots of white space
* Stick to **2–3 main colors** + neutral tones
* Use **modern fonts** (e.g., Poppins, Inter)
* Add **subtle animations** (hover effects, transitions)
* Keep UI simple and beginner-friendly

## Customization

You can easily customize:

* **Colors** → Edit in `style.css`
* **Content** → Update `index.html`
* **Animations** → Modify `script.js`

## Responsiveness

Make sure your design works on:

* Mobile phones
* Tablets
* Desktop screens

Use **media queries** in CSS for adjustments.
